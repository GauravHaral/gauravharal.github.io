Real-time Human Detection using Computer Vision

Developed the Human Detection and Counting System in this project using a Webcam, but you can use your own video or photographs. The following libraries were used:

  • OpenCV is a powerful machine learning library.
  • Imutils: Dedicated to Image Processing
  • Numpy is a scientific computing library. A numpy array is used to hold the image.
  • Argparse is a command-line input parser.
    • The input can be video, picture, or even a live camera in this model.

    Detection of Parkinson Disease

    Using an XGBClassifier, I created a model that can accurately detect the presence of Parkinson's disease in one's body. Loaded the data, extracted the features and labels, scaled the features, split the dataset, built an XGBClassifier, and the model had a 93.21 percent accuracy.

    Speech Emotion Recognition Project

    I learned to discern emotions from voice in this Machine Learning project. For this, I used an MLPClassifier, the soundfile library to read the sound file, and the librosa library to extract features from it. The model had a 74.4 percent accuracy rate.

    Deep Neural Network Based ChatBot

    In this project, I used NLTK, Keras, and Python to build a retrieval-based chatbot. The chatbot was trained using a dataset that included categories (intents), patterns, and responses. I utilized a specific recurrent neural network (LSTM) to classify the user's message and then chose a random answer from a list of possible responses.
